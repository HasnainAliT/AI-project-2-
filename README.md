# AI Project 2 - Data Classification Using AI

## Objective
Build a classification model using the Iris dataset.

## Technologies
- Python
- Scikit-Learn

## Workflow
1. Load Iris dataset
2. Apply StandardScaler
3. Split data into training and testing sets
4. Train KNN Classifier (K=5)
5. Evaluate using Accuracy, F1 Score and Confusion Matrix

## Run
pip install -r requirements.txt
python main.py
